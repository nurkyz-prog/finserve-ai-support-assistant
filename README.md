# FinServe AI Support Assistant

This project is a simple proof-of-concept AI solution for automating customer support in financial services.

## Problem
Customer support is handled manually, leading to slow responses and inconsistent communication.

## Solution
This prototype simulates an AI assistant that generates responses to common customer queries.

## Features
- Handles loan status questions
- Provides consistent responses
- Reduces manual workload

## Tech Stack
- Python
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
